This is an example implementation of the transaction.confirmed event using the socket event forwarder plugin created by friendsoflittleyus. This plugin is currently installed on Radians and is configured to fire the transaction.confirmed event at the 1st, 2nd, 3rd, 4th and 5th confirmation of a transaction. Developers can configure the amount of confirmations themselves according to their own usecase. In this example a customer(you) deposits tokens in an exchange. The exchange will add your funds to your exchange wallet after 5 network confirmations.
